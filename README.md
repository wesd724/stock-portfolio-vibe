# Stock Vibe

미국 주식 포트폴리오 시뮬레이션 웹 사이트

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19, TypeScript, Vite, Zustand, Recharts |
| Backend | NestJS, TypeScript, better-sqlite3 |
| 데이터 | yahoo-finance2, rss-parser |

## 실행 방법

```bash
# 클라이언트 (port 3000)
cd client
npm run dev

# 서버 (port 4000)
cd server
npm run start:dev
```

## 기능 소개

### 홈 화면

#### 시장 현황
- 주요 지수 및 자산 9종 실시간(15분 지연) 시세 카드 표시
  - 나스닥100 (NQ=F), S&P500 (ES=F), 코스피 (^KS11), 코스닥 (^KQ11)
  - 달러/원 환율 (KRW=X), 금 (GC=F), 은 (SI=F), 원유 (CL=F), 비트코인 (BTC-USD)
- 등락률 색상 구분 (상승: 초록, 하락: 빨강)

#### 국제 금융 뉴스
- Reuters, CNBC, Google News 3개 소스 뉴스 통합 제공
- 출처별 색상 배지 표시
- 10개씩 무한 스크롤
- 한국어 번역 토글 (Google Translate)
- 새로고침 버튼
- 기사 클릭 시 팝업 → 원문 링크 제공

---

### 종목 검색

- Navbar 상단 검색창에서 미국 주식(EQUITY) 및 ETF 검색
- 검색 결과 클릭 시 종목 정보 화면으로 이동 (어느 탭에서든 동작)

---

### 종목 정보

종목 검색 후 표시되는 상세 정보 화면

#### 종목 카드
- 현재가, 등락폭/등락률
- 장 상태 표시 (본장 / 프리장 / 애프터장 / 장마감)
- 프리장·애프터장 가격 및 변동률
- 당일 시가·고가·저가·거래량
- 시가총액, P/E, 배당수익률, 52주 범위
- 최신 업데이트(↻) 버튼으로 즉시 재조회
- 매수 버튼 → 매수 모달 오픈

#### 주가 차트
- Recharts 기반 라인 차트
- 인터벌: 1분 / 5분 / 1시간 / 1일
- 15분 지연 데이터

#### 종목 관련 뉴스
- Yahoo Finance 제공 뉴스 5건 + Google News 5건
- 한국어 번역 토글
- 새로고침 버튼
- 기사 클릭 시 팝업 → 원문 링크 제공

---

### 포트폴리오 (모의투자 + 백테스팅)

> 회원가입 없이 익명으로 동작하며, 데이터는 sessionStorage에 저장됩니다.
> (탭 닫으면 소멸, 새로고침은 유지)

#### 매수
- 종목 카드의 매수 버튼 클릭
- 과거 날짜 선택 가능 (1980년 이후)
- 투자 금액 입력 → 해당 날짜 실제 종가 조회 → 매수 주수 자동 계산
- 매수 확정 시 거래 내역 저장

#### 매도
- 사이드바 보유 종목의 매도 버튼 클릭
- 과거 날짜 선택, 수량 입력 (전량 버튼 제공)
- 해당 날짜 실제 종가 기준 매도 처리

#### 포트폴리오 탭
- 총 투자금 / 평가금액 / 평가손익 / 수익률 요약
- 보유 종목별 테이블 (보유수량, 평균단가, 현재가, 평가금액, 손익, 수익률)
- 종목 행 클릭 시 해당 종목 정보 화면으로 이동

#### 거래내역 탭
- 전체 매수/매도 내역 시간순 표시
- 거래 삭제 기능

#### 사이드바
- 보유 종목 목록 (평가금액, 수익률 실시간 표시)
- 종목 클릭 시 해당 종목 정보 화면으로 이동
- 현재가 새로고침 버튼
- 총 평가금액 / 수익률 요약 (UserInfo)

---

### 종목 비교

- 최대 10종목 동시 비교
- 종목 검색 후 추가, 칩 UI로 관리 (× 버튼으로 제거)
- 가격 변동률(%) 기준 정규화 차트 (첫 데이터 포인트 = 0%)
- 인터벌: 1분 / 5분 / 1일 / 한달
- 날짜 범위 직접 선택 가능 (인터벌 변경 시 자동 리셋)
- 10가지 색상으로 종목 구분
- 차트 위 종목명 클릭 시 해당 종목 정보 화면으로 이동

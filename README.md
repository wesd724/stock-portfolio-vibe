# Stock Vibe

미국 주식 포트폴리오 시뮬레이션 웹 사이트

[사이트 바로가기](http://stockvibe.p-e.kr)

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19, TypeScript, Vite, Zustand, Recharts |
| Backend | NestJS, TypeScript, better-sqlite3 |
| 데이터 | yahoo-finance2, rss-parser |

## 실행 방법

```bash
# 클라이언트 (port 3000)
cd client
npm run dev

# 서버 (port 4000)
cd server
npm run start:dev
```

## 기능 소개

### 홈 화면

**시장 현황 / 국제 뉴스** 서브탭으로 분리되어 있습니다.

#### 시장 현황
- 주요 지수·자산·통화 16종 실시간(15분 지연) 시세 카드 표시
  - 나스닥100 선물, S&P500 선물, 다우존스 선물, 러셀2000 선물
  - USD/KRW, 달러 인덱스, 미국 국채 2년, 미국 국채 10년, VIX 지수
  - 골드 선물, 은 선물, WTI 원유 선물
  - 코스피, 코스닥
  - 비트코인, 이더리움
- 카드 클릭 시 해당 종목 정보 화면으로 이동
- ? 버튼 클릭 시 지표 설명 모달 표시 (선물 / 달러 인덱스 / 국채 수익률 / VIX)
- **풋/콜 비율 (SPY · QQQ)**: 전체 만기 기준 풋콜 비율 표시, 별도 ? 버튼으로 설명 제공
- 새로고침 버튼

#### 국제 금융 뉴스
- Reuters, CNBC, Google News, AP News, MarketWatch 5개 소스 뉴스 통합 제공
- 출처별 색상 배지 표시
- 10개씩 무한 스크롤
- 한국어 번역 토글 (Google Translate)
- 새로고침 버튼
- 기사 클릭 시 팝업 → 원문 링크 제공

### 종목 정보

종목 검색 후 표시되는 상세 정보 화면

#### 종목 카드
- 현재가 및 본장 등락폭/등락률
- 장 상태 뱃지 표시 (본장 / 프리장 / 애프터장 / 장마감)
- 프리장·애프터장 가격 및 변동률 (데이터 있으면 항상 표시)
- **USD / KRW 전환**: 현재 환율 기준 원화 환산 가격 표시, 적용 환율 명시
- 당일 시가·고가·저가·거래량
- 시가총액, P/E, 배당수익률, 52주 범위
- 최신 업데이트(↻) 버튼, 즐겨찾기(★) 버튼
- 매수 버튼 → 매수 모달 오픈 (시장 현황 종목 — 지수·선물·환율·원자재·암호화폐 — 은 매수 불가)

#### 주가 차트
- Recharts 기반 라인 차트
- 인터벌: 1분 / 5분 / 15분 / 1시간 / 1일 / 1주 / 1달 / 12달
- 새로고침 버튼

#### 종목 관련 뉴스
- Yahoo Finance 제공 뉴스 5건 + Google News 5건
- 한국어 번역 토글
- 기사 클릭 시 팝업 → 원문 링크 제공

---

### 종목 목록 (스크리너)

- 미국 주식·ETF 실시간 목록 조회
- **종류 필터**: 전체 / 주식 / ETF
- **정렬 기준**:
  - 1일 등락률 (내림차순: 급등 종목, 오름차순: 급락 종목)
  - 1일 거래량
  - 3개월 평균 거래량
- **정렬 방향**: 오름차순 / 내림차순
- 종목 클릭 시 해당 종목 정보 화면으로 이동
- 본장 기준 · 15분 지연 데이터

---

### 종목 비교

- 최대 10종목 동시 비교
- 종목 검색 후 추가, 칩 UI로 관리 (× 버튼으로 제거)
- 가격 변동률(%) 기준 정규화 차트 (첫 데이터 포인트 = 0%)
- 인터벌: 1분 / 1시간 / 1일 / 1주 / 1달 / 12달
- 날짜 범위 직접 선택 가능 (인터벌 변경 시 자동 리셋)
- 10가지 색상으로 종목 구분

#### 하단 분석 섹션
- **기간 성과 요약**: 시작가·종가·기간 수익률 막대 그래프
- **지표 비교 테이블**: 현재가·등락률·시가총액·P/E·배당수익률·52주 최고/최저 나란히 비교
- **변동성 비교**: 수익률 표준편차 기반 리스크 크기 시각화
- **상관관계 매트릭스**: 종목 간 Pearson 상관계수 색상 히트맵 (2종목 이상 시 표시)

---

### 포트폴리오 (모의투자 + 백테스팅)

> 회원가입 없이 익명으로 동작하며, 데이터는 브라우저 로컬 스토리지에 저장됩니다.

#### 계좌 잔액
- 사이드바에서 초기 잔액 설정 (수정 버튼)
- 매수 시 잔액 차감, 매도 시 잔액 증가, 거래 삭제 시 자동 복구
- 잔액 부족 시 매수 차단 (추가 필요 금액 안내)
- KRW 모드에서 현재 환율 기준으로 환산 표시

#### 매수
- 종목 카드의 매수 버튼 클릭
- 과거 날짜 선택 가능 (1980년 이후, 주말 선택 시 거래 불가)
- 통화 선택: USD / KRW (종목 정보의 통화 설정 연동)
- **금액과 주수 동시 입력** (하나 입력 시 반대쪽 자동 계산)
- 숫자 입력 시 쉼표 자동 포맷팅
- 해당 날짜 실제 종가 + 환율(USDKRW) 동시 조회
- KRW 모드: 거래일 환율 기준 잔액 표시

#### 매도
- 사이드바 보유 종목의 매도 버튼 클릭
- 과거 날짜 선택 (최초 매수일 이전·주말 날짜 차단)
- **주수와 금액 동시 입력** (하나 입력 시 반대쪽 자동 계산)
- 전량 버튼 제공

#### 통화 표시 (KRW / USD 토글)
- 포트폴리오·거래내역·사이드바 전체에서 KRW ↔ USD 전환
- **KRW 보기**: 매수 시점 환율 기준 투자원금 + 현재 환율 기준 평가금액
- **환차손익 분리**: KRW 뷰에서 가격손익·환차손익 별도 표시
- 거래내역에 환율 컬럼 표시

#### 포트폴리오 탭
- 총 투자금 / 평가금액 / 평가손익 / 수익률 요약 (KRW/USD 전환)
- KRW 뷰: 가격손익 + 환차손익 컬럼 추가
- 보유 종목별 테이블 (보유수량, 평균단가, 현재가, 평가금액, 손익, 수익률)
- 종목 행 클릭 시 해당 종목 정보 화면으로 이동

#### 거래내역 탭
- 전체 매수/매도 내역 시간순 표시
- 환율 컬럼 표시
- 거래 삭제 기능 (확인 모달 후 삭제, 잔액 자동 복구)

### 투자 방식 시뮬레이터

과거 데이터를 기반으로 적립식·거치식 투자 전략의 수익률을 시뮬레이션합니다.

#### 적립식 투자 (DCA)
- 시작일, 종료일, 투자 주기(매일·매주·매달·매년), 1회 투자금액 설정
- 각 투자 시점의 실제 종가로 주식을 매수하는 방식으로 계산
- 총 투자원금 / 최종 평가금액 / 수익금 / 수익률 요약

#### 거치식 투자 (Buy & Hold)
- 매수일, 종료일, 투자금액 설정
- 매수일 종가에 전액 매수 후 보유 시 수익률 계산

#### 공통
- 평가금액 vs 투자원금 Area 차트 (Recharts)
- 종목 검색으로 어떤 미국 주식·ETF든 시뮬레이션 가능
- 시뮬레이션 요약 정보 표시

---

### 즐겨찾기

- 종목 정보 화면의 ★ 버튼으로 추가/해제
- 즐겨찾기 탭에서 저장된 종목의 현재가·등락·장 상태 요약 표시
- 행 클릭 시 해당 종목 정보 화면으로 이동
- 그룹 추가·이름 변경·삭제, 그룹 간 종목 이동
- 순서 변경: ▲▼ 버튼으로 종목 순서 및 그룹 순서 변경 (PC/모바일 동일)
- 브라우저 로컬 스토리지에 저장 (새로고침·재방문 시 유지)

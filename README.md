# Stock Vibe

미국 주식 포트폴리오 시뮬레이션 웹 사이트

## 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React 19, TypeScript, Vite, Zustand, Recharts |
| Backend | NestJS, TypeScript, better-sqlite3 |
| 데이터 | yahoo-finance2, rss-parser |

## 실행 방법

```bash
# 클라이언트 (port 3000)
cd client
npm run dev

# 서버 (port 4000)
cd server
npm run start:dev
```

### 종목 정보

종목 검색 후 표시되는 상세 정보 화면

#### 종목 카드
- 현재가 및 본장 등락폭/등락률 (명시적 "본장" 라벨)
- 장 상태 표시 (본장 / 프리장 / 애프터장 / 장마감)
- 프리장·애프터장 가격 및 변동률 (데이터 있으면 항상 표시)
- 당일 시가·고가·저가·거래량
- 시가총액, P/E, 배당수익률, 52주 범위
- 최신 업데이트(↻) 버튼으로 즉시 재조회
- 매수 버튼 → 매수 모달 오픈

#### 주가 차트
- Recharts 기반 라인 차트
- 인터벌: 1분 / 5분 / 1시간 / 1일
- 15분 지연 데이터
- 새로고침 버튼

#### 종목 관련 뉴스
- Yahoo Finance 제공 뉴스 5건 + Google News 5건
- 한국어 번역 토글
- 새로고침 버튼
- 기사 클릭 시 팝업 → 원문 링크 제공

### 종목 목록 (스크리너)

- 미국 주식·ETF 실시간 목록 조회
- **종류 필터**: 전체 / 주식 / ETF
- **정렬 기준**:
  - 1일 수익률 (내림차순: 급등 종목, 오름차순: 급락 종목)
  - 1일 거래량
  - 3개월 평균 거래량
- **정렬 방향**: 오름차순 / 내림차순
- 종목 클릭 시 해당 종목 정보 화면으로 이동
- 본장 기준 · 15분 지연 데이터

### 포트폴리오 (모의투자 + 백테스팅)

> 회원가입 없이 익명으로 동작하며, 데이터는 sessionStorage에 저장됩니다.
> (탭 닫으면 소멸, 새로고침은 유지)

#### 매수
- 종목 카드의 매수 버튼 클릭
- 과거 날짜 선택 가능 (1980년 이후)
- 통화 선택: USD / KRW
- **금액과 주수 동시 입력** (하나 입력 시 반대쪽 자동 계산)
- 숫자 입력 시 쉼표 자동 포맷팅
- 해당 날짜 실제 종가 + 환율(USDKRW) 동시 조회

#### 매도
- 사이드바 보유 종목의 매도 버튼 클릭
- 과거 날짜 선택 (최초 매수일 이전 날짜 차단 — 달력·직접입력 모두)
- **주수와 금액 동시 입력** (하나 입력 시 반대쪽 자동 계산)
- 전량 버튼 제공
- 해당 날짜 실제 종가 + 환율 표시

#### 통화 표시 (KRW / USD 토글)
- 포트폴리오·거래내역·사이드바 전체에서 KRW ↔ USD 전환
- **KRW 보기**: 매수 시점 환율 기준 투자원금 + 현재 환율 기준 평가금액
- **환차손익 분리**: KRW 뷰에서 가격손익·환차손익 별도 표시
- 거래내역에 환율 컬럼 표시

#### 포트폴리오 탭
- 총 투자금 / 평가금액 / 평가손익 / 수익률 요약 (KRW/USD 전환)
- KRW 뷰: 가격손익 + 환차손익 컬럼 추가
- 보유 종목별 테이블 (보유수량, 평균단가, 현재가, 평가금액, 손익, 수익률)
- 종목 행 클릭 시 해당 종목 정보 화면으로 이동

#### 거래내역 탭
- 전체 매수/매도 내역 시간순 표시
- 환율 컬럼 표시
- 거래 삭제 기능

#### 사이드바
- 보유 종목 목록 (평가금액, 수익률 실시간 표시, KRW/USD 전환)
- 종목 클릭 시 해당 종목 정보 화면으로 이동
- 현재가 새로고침 버튼
- 총 투자금 / 평가금액 / 수익률 요약 (KRW/USD 토글)

### 종목 비교

- 최대 10종목 동시 비교
- 종목 검색 후 추가, 칩 UI로 관리 (× 버튼으로 제거)
- 가격 변동률(%) 기준 정규화 차트 (첫 데이터 포인트 = 0%)
- 인터벌: 1분 / 5분 / 1일 / 한달
- 날짜 범위 직접 선택 가능 (인터벌 변경 시 자동 리셋)
- 10가지 색상으로 종목 구분
